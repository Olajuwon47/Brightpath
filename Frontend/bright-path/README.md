# BrightPath Platform

Welcome to the **BrightPath Platform**! This README is designed to help **everyone**—from HR to non‑technical team members—get the project up and running quickly, understand what it does, and see how the pieces fit together.

---

## 🚀 Overview

BrightPath is a modern web application for managing school operations:  

- Student and teacher profiles  
- Course and class scheduling  
- Attendance tracking  
- Reporting dashboards  

It’s built with **Next.js** for a fast, SEO‑friendly frontend, uses **Framer Motion** for smooth animations, and leverages **shadcn/ui** for a consistent, accessible design system.

## 🔧 Prerequisites

Before you begin, make sure you have:

- **Node.js** (LTS version recommended)  
  Download and install from: <https://nodejs.org/>  
- **npm** (comes bundled with Node.js)  

---

## 🏁 Getting Started

1. **Clone the repository**  

   ```bash
   git clone https://github.com/your‑org/tpn‑schools.git
   ```

2. **Change into project directory**  

   ```bash
   cd tpn‑schools
   ```

3. **Install dependencies**  

   ```bash
   npm install
   ```

4. **Start the development server**  

   ```bash
   npm run dev
   ```

5. **Open in your browser**  
   Visit <http://localhost:3000>

## 🤝 How It Works (Non‑Technical)

- **Next.js**: Think of this as the engine and frame of a car—it delivers pages fast and handles navigation without full page reloads.  
- **Framer Motion**: Adds polish by animating elements (menus sliding in, buttons fading, etc.).  
- **shadcn/ui**: A library of ready‑made, styled components (cards, buttons, forms) so our pages look consistent and professional.
